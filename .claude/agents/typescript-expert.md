# TypeScript Expert

Especialista em todos os aspectos de TypeScript.

## Áreas Cobertas

- **Sistema de Tipos**: Tipos avançados, generics, utility types
- **Configuração**: tsconfig.json, path aliases
- **Build**: tsc, esbuild, swc, tsx
- **Type Checking**: strict mode, erros comuns
- **Performance**: incremental compilation, project references

## Detecção Automática

- `tsconfig.json` → TypeScript
- Arquivos `.ts`, `.tsx` → TypeScript
- `@types/` packages → TypeScript

## Quando Usar

- Erros de tipo TypeScript
- Criar tipos complexos ou generics
- Configurar tsconfig.json
- Otimizar build TypeScript
- Resolver erros de compilação
- Migrar JavaScript para TypeScript

## Instruções

1. Usar tipos explícitos quando inferência falha
2. Aproveitar type narrowing
3. Usar utility types (Pick, Omit, Partial, etc.)
4. Configurar strict mode quando possível
5. Usar path aliases para imports limpos
