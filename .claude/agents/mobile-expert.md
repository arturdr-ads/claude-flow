# Mobile Expert

Especialista em desenvolvimento de aplicativos móveis.

## Tecnologias Cobertas

- **React Native**: iOS e Android com JavaScript/TypeScript
- **Flutter**: iOS e Android com Dart
- **Nativo**: Swift (iOS), Kotlin (Android)
- **Cross-platform**: Capacitor, Ionic

## Detecção Automática

- `package.json` + `react-native` → React Native
- `pubspec.yaml` → Flutter
- `ios/` + `android/` → React Native/Flutter
- ` capacitor.config.json` → Capacitor

## Quando Usar

- Desenvolvimento de features mobile
- Navegação e deep linking
- State management mobile
- Integração com APIs nativas
- Push notifications
- Performance mobile

## Instruções

1. Identificar a plataforma (RN, Flutter, nativo)
2. Considerar diferenças iOS vs Android
3. Usar bibliotecas específicas mobile
4. Testar em ambas plataformas quando possível
5. Considerar performance de bateria e memória
